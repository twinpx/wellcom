Rehab Fitness Matrix BASE 1 — c 24 по 26 октября

Программа Rehab Fitness Matrix представляет собой уникальное сочетание - персональный подход к тренировкам клиентов с учетом особенностей их осанки, реализуемый в групповом формате. Базовый курс Rehab Fitness Matrix – это первая ступень освоения знаний, необходимых для применения в групповых программах принципов и методик, используемых в персональном формате коррекционных тренировок.

Записаться на Rehab

От фитнес-тренера, получающего сертификацию Инструктора RFM 1 уровня, требуется знание базовых коррекционных упражнений, умение корректировать у клиентов технику выполнения этих упражнений, знание технологии определения осанки и проведения фитнес-тестирования для контроля результатов, а также четкое исполнение инструкций по поэтапному внедрению программы.

Дизайн программ, музыкальное сопровождение, последовательность проведения занятий в недельном цикле – все составляющие программы заложены в технологию, которую надо лишь неукоснительно выполнять.

Обучение на Базовом курсе 1 уровня (BASE 1) Rehab Fitness Matrix позволяет фитнес-тренеру проводить коррекционные фитнес-занятия с базовыми знаниями по функциональной анатомии, физиологии, методике фитнес-тренировки.

В программе дистанционного курса (теория):

  • Описание, Цели и Средства уроков Rehab Fitness Matrix
  • Технологии, методики и оборудование RFM
  • Методика определение состава тела (биоимпедансный метод или каллиперометрия)
  • Определение сколиоза и одного из 4-х видов осанки с помощью двигательных тестов на сетке «RFM Тест Осанка», определение МПК
  • В программе очного курса (практика):
  • Проведение тестирования МПК
  • Проведение тестирования осанки
  • Проведение тестирования состава тела
  • Техника выполнения упражнений RFM 1 уровень
  • Правила корректировки техники у клиентов
  • Речевые модули для объяснения клиентам техники выполнения и ее коррекции
  • Мастер-классы RFM Balance, RFM Core, RFM Flex (1 уровень)

На семинаре вы научитесь:

  • определять нарушение осанки и распределять клиентов по 4-м основным видам осанки
  • определять сколиоз
  • проводить уроки RFM 1 уровня под заданный дизайн (последовательность и музыкальное сопровождение)
  • корректировать технику выполнения упражнений относительно вида нарушения осанки
  • применять методики миофасциального расслабления, функциональной тренировки по мышечным цепям, коррекционные и дыхательные упражнения

Пакет участника обучения:

  • Презентации
  • Методическое пособие
  • Каталог упражнений
  • Последовательность упражнений в уроке с количеством подходов и повторений (дизайн уроков) для уроков RFM Balance, RFM Core, RFM Flex (1 уровень)
  • Музыкальные треки (55 мин) RFM Balance, RFM Core, RFM Flex (1 уровень)
  • Сертификат при успешном прохождении теоретического и практического блоков
